# deep-learning-cnn-project
# CNN-based image classification project using TensorFlow/Keras for malaria detection

## Overview
This project implements a Convolutional Neural Network (CNN) to detect malaria from cell images. The model is trained to classify infected and uninfected samples.

## Features
- Data preprocessing
- CNN model implementation
- Model training and evaluation
- Performance visualization

## Tech Stack
- Python
- TensorFlow / Keras
- Google Colab

## Files
- cnn_final.ipynb – main notebook

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
